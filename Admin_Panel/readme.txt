# NatureCeylon Admin Panel

This is the admin panel for the NatureCeylon project.

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/NatureCeylon.git
    ```
2. Navigate to the project directory:
    ```
    cd NatureCeylon_Struct/Admin_Panel
    ```
3. Install dependencies:
    ```
    composer install
    npm install
    ```

## Usage

1. Start the development server:
    ```
    php artisan serve
    ```
2. Open your browser and navigate to `http://localhost:8000`.
